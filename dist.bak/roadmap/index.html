Roadmap

The planned development path for OpenCS. Topics are prioritized based on curriculum relevance and pedagogical value.

Milestone 1 — Non-Topic Shell

Planned

Landing page, topic index, topic layout, contributor pages, navigation, and design system.

  • Landing page with project introduction
  • Topic index with category filtering
  • Topic layout template with pedagogy blocks
  • Contributor pages (contribute, about, roadmap)
  • Design system tokens and components

Milestone 2 — Engine Implementations

Planned

Four reusable engines that power all topic visualizations.

  • Sequence Engine — linear operations, arrays, sorting
  • TreeGraph Engine — trees, BSTs, graphs, traversal
  • Theory Engine — DFA/NFA, automata theory
  • SystemProcess Engine — CPU scheduling, process management

Milestone 3 — First Topic Modules

Planned

Ship the first four complete topics with working visualizers.

  • Sorting Visualizer (bubble, merge, quick)
  • BFS/DFS Graph Traversal
  • DFA/NFA Simulator
  • CPU Scheduling (FCFS, SJF, Round Robin)

Milestone 4 — Remaining Topics

Planned

Complete the initial 12-topic set.

  • Recursion Tree Visualizer
  • Time Complexity Growth
  • Array Memory Operations
  • Stack and Queue Simulator
  • Linked List Visualizer
  • Tree Traversal Explorer
  • BST Insert/Delete/Search
  • Shortest Path and MST
